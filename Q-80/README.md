# Rotate Character

برنامه ای بنویسید که به تعداد گفته شده، کاراکتر های یک رشته را بچرخاند!

**ورودی:**

در خط اول ورودی، به شما رشته داده خواهد شد و در خط دوم به شما عددی مثبت داده خواهد شد.

**خروجی:**

در تنها خط خروجی جواب را چاپ کنید!

## Input-1:
```
amir
4
```

## Output-1:
```
amir
```

## Input-2:
```
amir
5
```

## Output-2:
```
mira
```

## Input-3:
```
amir
543
```

## Output-3:
```
rami
```
