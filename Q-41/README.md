# Symmetric List

برنامه ای بنویسید که لیستی از پارامترها را دریافت کند و چک کند آیا پارامترهای ارسالی symmetric هست یا خیر.
symmetric به لیستی میگویند که از دو طرف با هم بر ابر باشند.

### Examples:

**Input1:**
```
[1,2,3,4,5,4,3,2,1]
```

**Output1:**
```
True
```

**Input2:**
```
[1,2,3,4,5,6,7]
```

**Output2:**
```
False
```
