# مبنای فیبوناچی:

برنامه ای بنویسید که عددی را از ورودی دریافت کند و عدد را در مبنای فیبوناچی چاپ کند.

عدد در مبنای فیبوناچی عددی است که از اعداد فیبوناچی کوچکتر از خودش ایجاد میشود.

**Input:**
```
101
```

**Output:**
```
104 = 89 + 13 + 2
```

**Input:**
```
456
```

**Output:**
```
456 = 377 + 55 + 21 + 3
```
