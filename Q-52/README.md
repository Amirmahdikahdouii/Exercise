# کار با ماتریس و اعداد دودویی

برنامه ای بنویسید که یک ماتریس دلخواه از اعداد طبیعی را بخواند و آن را تبدیل به یک ماتریس دودویی کند.

## Example:

**Input:**
```
Please Enter Records Count: 2
Please Enter Sells Count: 2
Sell Number: 2
Sell Number: 10
---------------
Sell Number: 20
Sell Number: 30
---------------
```

**Output:**

```
[[2, 10], [20, 30]]
[['10', '1010'], ['10100', '11110']]
```
