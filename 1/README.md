استاد پارسا از او خواسته تا تعداد اعداد اول n رقمی را پیدا کند.
پارسا حوصله عملیات جبری روی کاغذ را ندارد و از شما کمک خواسته تا برایش این مسئله را حل کنید.
# ورودی:
<code>
    ورودی عددی صحیح n , 
    <br>
    0 < n < 6
</code>

</code>
